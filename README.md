# ☕ Starbucks Sales & Nutrition Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Starbucks Sales & Nutrition Dashboard** developed using **Microsoft Power BI**. The dashboard provides comprehensive insights into Starbucks' beverage categories, nutritional values, caffeine content, and beverage distribution through interactive visualizations.

The objective of this project is to transform Starbucks beverage data into meaningful business insights that support product analysis and customer decision-making.

---

# 🎯 Business Objective

To analyze Starbucks beverage data by evaluating nutritional values, beverage categories, caffeine levels, and product distribution using interactive dashboards and KPI-driven visualizations.

The dashboard helps answer key business questions such as:

* Which beverage categories contain the highest calories?
* Which drinks have the highest caffeine content?
* What is the distribution of beverage categories?
* How do nutritional values vary across beverage types?
* How can customers make healthier beverage choices?

---

# 📊 Key Performance Indicators (KPIs)

| KPI              | Description                            |
| ---------------- | -------------------------------------- |
| Total Beverages  | Total number of beverages available    |
| Average Sugar    | Average sugar content across beverages |
| Average Calories | Average calories per beverage          |
| Average Caffeine | Average caffeine content               |

# 📈 Dashboard Features

## 1️⃣ Average Calories by Beverage Category

**Objective:** Compare calorie content across beverage categories.

**Visualization:** Area Chart

---

## 2️⃣ Starbucks Beverage Category Distribution

**Objective:** Analyze the distribution of available beverage categories.

**Visualization:** Donut Chart

---

## 3️⃣ Starbucks Global Beverage Presence

**Objective:** Compare the availability of beverages across different categories.

**Visualization:** Column Chart

---

## 4️⃣ Average Caffeine by Category

**Objective:** Identify beverage categories with the highest average caffeine levels.

**Visualization:** Horizontal Bar Chart

---

## 5️⃣ Top 5 Highest Caffeine Beverages

**Objective:** Highlight beverages containing the highest caffeine content.

**Visualization:** KPI Cards

---

## 6️⃣ Interactive Filters

Users can dynamically explore the dashboard using:

* Beverage Preparation Type
* Protein Range
* Interactive Cross Filtering
